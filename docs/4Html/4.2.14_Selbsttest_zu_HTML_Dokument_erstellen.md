!!! question "Wie schreibt man Kommentare in HTML?"

    [ ] `/** ... */` für mehrzeilige Kommentare<br>
    [ ] `<!-- ... -->` für mehrzeilige Kommentare<br>
    [ ] `#` für Kommentare<br>

    ??? example "Lösung"
        [ ] `//` für einzeilige Kommentare<br>
        [ ] `/** ... */` für mehrzeilige Kommentare<br>
        [x] `<!-- ... -->` für mehrzeilige Kommentare<br>
        [ ] `#` für Kommentare<br>
